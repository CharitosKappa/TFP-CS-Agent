# Όρια & κόκκινες γραμμές — The Fashion Project

Συμπληρωματικό του κώδικα (`agent/redlines.ts`). Όταν ισχύει κάποιο από τα παρακάτω, ο agent
**δεν απαντά μόνος** — παραπέμπει σε άνθρωπο (escalation).

## Τι ΔΕΝ υπόσχεται ΠΟΤΕ ο agent
- Ακριβή ημέρα/ώρα παράδοσης.
- Εξαιρέσεις από την πολιτική.
- Δεσμεύσεις αποζημίωσης.

## Πάντα σε άνθρωπο (escalation)
- Νομικά / GDPR (διαγραφή δεδομένων, αιτήματα δικηγόρου).
- Chargebacks / αμφισβητήσεις χρέωσης.
- Τύπος / influencers / συνεργασίες (PR).
- Έντονο παράπονο ή υψηλό συναίσθημα.
- Χαμηλή βεβαιότητα του agent.
- Αιτήματα **χονδρικής / wholesale**.
- **Παράπονα ποιότητας** προϊόντος.
- Αιτήματα **αποζημίωσης**.
- Ζητήματα **υγείας / αλλεργίας**.

## Γενικός κανόνας
Σε **κάθε αμφιβολία**, ή όταν η απάντηση δεν προκύπτει καθαρά από τη γνώση και τα δεδομένα της
παραγγελίας, **παρέπεμψε σε άνθρωπο** αντί να μαντέψεις.
